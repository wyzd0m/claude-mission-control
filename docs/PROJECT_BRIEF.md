# Approved Project Brief

> Provided by the project owner on 2026-07-05. This brief is consistent with the existing planning
> documents and introduces no scope changes.

Claude Mission Control is a local-first MCP workspace extension for Claude Desktop.

Claude Desktop remains the AI interface, while Mission Control owns structured project state,
including tasks, development stages, decisions, blockers, artifacts, checkpoints, and observable
activity events.

The primary interface combines:

- A clean futuristic facility in a low-poly isometric style
- Small robots dispatched into department rooms when Mission Control tools are used
- A conventional activity panel showing the exact tool, status, progress, result, or error
- A project-stage indicator
- A recent-event timeline
- Accessible controls that remain usable without the 3D visualization

The facility visualizes only real Mission Control events and saved project state. It does not claim
to expose Claude's hidden reasoning, complete conversation history, or unrelated tool activity.

Version 1 will be generated by Claude, use procedurally created 3D geometry, require no Anthropic
API key, store data locally, and ship as a one-click Claude Desktop extension.
